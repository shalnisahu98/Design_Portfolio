# 💼 Shalni's Developer Portfolio

Welcome to **Shalni's Developer Portfolio Website** — a personal web-based resume designed to showcase the skills, experience, and projects of Shalni, an aspiring web developer from India.

## 📝 Overview

This is a responsive and modern personal portfolio built with **HTML**, **CSS**, and **Font Awesome**. It highlights Shalni's background, development skills, UI/UX services, and project work. It's suitable for sharing with potential employers, clients, or collaborators.

## 🚀 Features

- **Responsive Design** for mobile and desktop
- **Navigation Menu** with toggle for smaller screens
- **About Section** with tabs for Skills and Education
- **Services Section** outlining key offerings (Web Design, UI/UX, App Design)
- **Portfolio Gallery** with project previews
- **Contact Form** for inquiries
- **Social Media Integration**
- **Resume Download Button**

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (for interactivity)**
- **Font Awesome** (for icons)
